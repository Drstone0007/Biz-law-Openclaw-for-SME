## ⚙️ How It Works

1. User visits website  
2. Fills the lead form  
3. Form submits to backend  
4. Backend processes data  
5. System triggers:

   - 📧 Email → sent to your inbox  
   - 📲 WhatsApp → instant alert  
   - 🧾 Logs → stored for tracking
